# OUMI — National Digital Platform for Women's and Maternal Health in Chad

## 📋 Project Overview
OUMI is an innovative technological solution designed to transform access to maternal healthcare in Chad. By connecting patients, doctors, and hospitals within a secure ecosystem, OUMI aims to reduce maternal mortality through early risk detection and the modernization of medical tracking.

---

## 🚀 Technological Architecture (Premium Start-up Level)
The project relies on a cutting-edge tech stack chosen for its robustness, scalability, and ability to operate in environments with limited connectivity.

### ⚛️ Frontend: Flutter (Dart)
* **Architecture**: **Feature-First Clean Architecture**. The code is organized by features (`auth`, `pregnancy`, `appointments`, etc.), making maintenance and updates easier.
* **State Management**: **Riverpod**. Utilizes Flutter's most modern state management for smooth responsiveness and testable code.
* **Navigation**: **GoRouter**. Advanced management of routing and deep linking.
* **Design System**: Inspired by Figma, utilizing the **Plus Jakarta Sans** font and a custom Material 3 theme (Teal & Pastel).

### ☁️ Backend: Firebase
* **Authentication**: Secure account management for women, fathers, and doctors.
* **Cloud Firestore**: Real-time NoSQL database for tracking health indicators.
* **Cloud Storage**: Secure storage for medical documents and ultrasounds.

---

## 🛠️ Innovative Implemented Features
1. **Risk Assessment Engine**: An intelligent questionnaire that classifies health status using a color-coded system (Green/Yellow/Red) based on medical protocols.
2. **OUMI Voice Assistant**: Integration of speech-to-text and text-to-speech (STT/TTS) to ensure inclusion for illiterate individuals (French/Chadian Arabic).
3. **Escrow System (Secure Payment)**: Simulates holding funds via Airtel/Moov Money, which are only released after the consultation is validated.
4. **QR Code Certification**: Generates digital signatures to authenticate health records with partner clinics.
5. **Intelligent Emergency Module**: Selects the nearest hospital, visualizes available rooms, and provides a GPS ambulance request button.

---

## 📉 Social Impact & Innovation
* **Digital Inclusion**: The first Chadian healthcare tool natively designed for local languages through voice interaction.
* **Real-Time Availability**: Reduces hospital overcrowding through immediate visibility of reception capacity.
* **Structured Data**: Supports decision-making for health authorities through anonymized statistical dashboards.
